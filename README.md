# GR21-ML-Classification

Machine Learning Classification project developed for data analysis and predictive modeling using classification algorithms.
This project focuses on training, evaluating, and comparing machine learning models for accurate classification tasks.

## Features

* Data preprocessing and cleaning
* Training multiple ML classification models
* Model evaluation and performance metrics
* Visualization of results
* Easy-to-understand project structure

## Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Project Structure

```bash
GR21-ML-Classification/
│
├── data/               # Dataset files
├── notebooks/          # Jupyter notebooks
├── models/             # Saved trained models
├── src/                # Source code
├── results/            # Evaluation results and visualizations
├── requirements.txt    # Project dependencies
└── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/ElsaKrasniqii/GR21-ML-Classification.git
```

Navigate to the project folder:

```bash
cd GR21-ML-Classification
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the training script:

```bash
python train.py
```

Run model evaluation:

```bash
python evaluate.py
```

Or open the Jupyter notebooks:

```bash
jupyter notebook
```
